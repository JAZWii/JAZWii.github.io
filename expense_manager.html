Expense Manager App.

Expense Manager is a cross-platform application that work on both iOS and Android which was developed using Flutter from scratch, Expense Manager App can present a statistic of your daily, weekly monthly, annually and overall expenses, so you can meet your saving goals, it can show them using charts and easy to understand graphs.

  • Language/s: Flutter, Dart
  • Repository: Github
  • Category: Mobile Application, iOS, Android
