2D Platformer Microgame

Platformer Microgame is a classic 2D platform game that you can mod and make your own Creative Mods to tweak the project and add your own levels, while learning the basics of Unity. The current Game it is done by following Unity's Tutorial in addition to my own touches. The Development was done using Unity 2019.2.19
