FPS Microgame

The FBS (First Person Shooter) Microgame is packed with versatile level building assets, weapons, props, enemies, and more. The FBS a First Person Shooter game that was created as a Template by Unity for learning purposes. The current Game it is done by following Unity's Tutorial in addition to my own touches. The Development was done using Unity 2019.2.19
